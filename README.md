Reactive Stream Ruby
==========================

Reactive Streams for Ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'reactive-streams-rb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install reactive-streams-rb
